# Android Firebase files by flavor

Place one `google-services.json` per flavor:

- `android/app/src/prod/google-services.json` for Firebase project `planerz`
- `android/app/src/preview/google-services.json` for Firebase project `planerz-preview`

Expected Android application IDs:

- `prod` -> `fr.centuryspine.planerz`
- `preview` -> `fr.centuryspine.planerz.preview`

Template files are provided:

- `android/app/src/prod/google-services.json.example`
- `android/app/src/preview/google-services.json.example`
