# Notes de version — v0.3.1

> Période couverte : v0.3.0 → v0.3.1

---

## Analyse du delta (avant rédaction)

Sur la fenêtre `v0.3.0 → v0.3.1`, le delta livré est centré sur la prévention des suppressions involontaires. Le produit ajoute une étape de confirmation avant suppression dans les flux **Repas** et **Liste de courses**. Aucun autre domaine fonctionnel n’évolue dans cette release.

---

## 1) L'essentiel pour les utilisateurs

### Repas

- Une confirmation est maintenant demandée avant de supprimer un **plat** dans la vue des repas cuisinés.
- Une confirmation est maintenant demandée avant de supprimer un élément **potluck**.

### Liste de courses

- Une confirmation est maintenant demandée avant de supprimer un article depuis le menu d’une ligne.
